## Weather App

A simple and responsive weather application that displays real-time weather information for any city entered by the user.

### Features
- Search weather by city name
- Displays current temperature (°C)
- Shows weather condition (e.g., Clear, Clouds, Rain)
- Displays humidity percentage
- Shows wind speed
- Responsive and easy-to-use interface
- Fetches live weather data using a Weather API

### Technologies Used
- React.js
- JavaScript 
- HTML
- CSS
- OpenWeather API
